# 2_技术与研发 - AGENTS 指南

## 👤 用户画像映射与基调设定
* **画像标签**：10年资深大前端开发工程师。
* **技术栈深度基底**：
  * **基础与框架**：精通 HTML/CSS/JS/TS/ES6/Sass/Less。精通 Vue2/3 全家桶 (Vue Router/Vuex/Pinia/Vue CLI/Vite/Webpack)。了解 React 生态 (React Router/Hooks/Redux)。
  * **生态与依赖**：熟悉 Vue 生态常用依赖 (Axios/Echarts/ElementUI/Ant Design)。
  * **多端与跨平台**：精通响应式设计、移动端 H5、React Native、uni-app、微信小程序。了解 PWA、Electron。
  * **后端与运维**：精通 Git。了解 Nginx、Docker、Jenkins、Kubernetes。熟悉前后端分离与 RESTful API 交互规范。
  * **全栈视野**：了解 Node.js、Nest.js、Express.js 及其相关的数据库、缓存、消息队列等后端生态。
  * **工程化与架构**：擅长独立封装工具和业务组件。了解微前端、低代码开发。
* **内容红线（绝对禁止）**：绝不记录“一查就有”的纯 API 语法、基础概念教程、环境搭建类水文。
* **核心价值（必须聚焦）**：体系化的回顾与学习。重点聚焦：体系化、经验、选型、踩坑、最佳实践、工程思路、架构设计。
* **AI 交互要求**：10年前端老炮视角。语言专业、精炼、有深度，拒绝啰嗦和堆砌概念。

## 📂 子目录详细规范与体系化演进方向

### 21_前端开发 (核心纵深区)
*聚焦大前端领域的基础稳固、框架底层及工程化架构体系。*
* **211 语言底座与高阶特性**：ES6+ 进阶、TS 高级类型体操、闭包/原型链深层机制、性能基准测试。
* **212 框架内核与生态**：Vue2/3 响应式原理深度解构、Vite/Webpack 构建优化、React 生态核心概念对比、状态管理最佳实践。
* **213 组件化与低代码**：高复用业务组件封装、定制化 UI 组件库设计、低代码引擎核心逻辑设计与实现。
* **214 性能优化与安全**：首屏秒开优化方案、长列表渲染极限优化、前端防攻击 (XSS/CSRF)、前端监控埋点设计。
* **215 跨端与多端底层架构**：React Native 与 uni-app 架构选型、微信小程序底层机制、响应式适配方案、PWA/Electron 探索应用。

### 22_后端开发与中间件 (全链路拓展区)
*立足全栈工程师视角，打通前后端数据链路与底层架构。*
* **221 服务端语言与框架**：Node.js 底层模型、Nest.js/Express.js 框架设计、中间件洋葱模型、BFF 层高并发聚合设计。
* **222 关系型数据库**：前端视角的 MySQL 表结构设计、慢查询调优、索引防踩坑。
* **223 非关系型数据库与缓存**：Redis 核心数据结构、接口防刷防击穿策略、MongoDB 实战。
* **224 消息队列与中间件**：基于 Node.js 操作 RabbitMQ/Kafka 进行异步任务削峰、消息防丢失设计。

### 23_系统架构与设计模式 (顶层设计区)
*系统级思考，脱离具体业务代码的工程重构与架构抽象能力。*
* **231 设计模式与重构**：前端复杂业务场景（如低代码平台、BI引擎）下的设计模式演进、SOLID 原则在定制化组件库中的落地。
* **232 分布式与微前端架构**：微前端 (qiankun/wujie) 架构拆分与沙箱隔离实战、后端微服务的前端网关层映射。
* **233 业务系统设计实战**：RESTful API 规范体系设计、中后台复杂权限 (RBAC) 全链路闭环、定制化业务线组件抽象与组合设计。

### 24_运维部署与云原生 (DevOps 基建区)
*保障工程稳定、可伸缩、全自动化的基础设施流转。*
* **241 操作系统与网络**：Linux 高级调错、HTTP协议、前后端分离鉴权机制 (JWT/OAuth2)。
* **242 容器化与服务编排**：Docker 镜像瘦身实践、Kubernetes (K8s) Pods 与节点在前端高可用部署中的认知。
* **243 自动化流水线 CI-CD**：GitLab CI、Jenkins 自动化构建与测试集成、GitHub Actions 工作流配置。
* **244 服务器运维与监控**：Nginx 反向代理与负载均衡高阶配置、前端静态资源缓存调优、服务器资源性能压测与监控。

### 25 AI 与前沿探索 (技术嗅觉区)
*保持技术敏锐度，跟进能提升工程效能的前沿基建。*
* **251 大模型原理与开发**：LLM 基础概念、RAG 检索增强生成原理及前端展示调优。
* **252 Prompt 提示词工程**：开发效能提升的结构化 Prompt 沉淀、辅助 CR 提示词。
* **253 AI 辅助编程与 Agent**：Cursor/Copilot 在复杂重构中的进阶玩法、个人知识库 Agent 化体系搭建。
* **254 前沿技术涉猎**：WebAssembly、Rust 等边缘技术的持续追踪。

### 26_解决方案与代码片段 (Snippets & Bugfix 避坑区)
*纯实战的碎片化记录隔离区，所有零碎实战、开箱即用的模块都沉淀在这里。*
* **261 报错排查与踩坑记录**：记录特定场景的疑难杂症，例如 `Uni-app 混合开发内存溢出排查.md`。必须包含现象、根因和解决方案。
