## 🛠️ 方法论映射
| 产品概念 | 前端映射 | 核心关注点 |
| :--- | :--- | :--- |
| **User Flow** | 路由图 (Router Graph) | 每一个跳转都是一个 `pushState` 或 `replaceState` |
| **MVP** | 首屏渲染 (First Paint) | 先跑通核心逻辑 HTML 骨架，再迭代 CSS 样式 |
| **Information Architecture** | 组件树与状态树 | 父子组件传参清晰，数据流向明确 |
| **Fitts's Law** | 交互热区 (Padding/z-index) | 点击目标越近、越大，`click` 成本越低 |
| **异常流设计** | Error Boundary / Promise Catch | 处理空状态 (Empty)、加载中 (Loading)、报错 (Error) |
